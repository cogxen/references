1. Top 10 Most Home/Away Team Matchups¶

Let's count the number of matches in each home/away matchup.

  • group results by home_team and away_team
  • use the count aggregation to count the date column -- this will count the number of games in each group

We've added some code to clean the output and then display the top 10 most frequent matchups. What do you notice?

matchups = results.groupby(['home_team', 'away_team']).agg({'date': 'count'})

# clean the groupby
matchups.columns = ['num_games']
matchups = matchups.reset_index()
